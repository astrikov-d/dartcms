Loads new content for the popup from the server.
load(options)
options object, options for the popup

Description

The first argument options is a map of properties for the popup (see overview page for valid options). The options argument must have url property defined, which will be called to retrieve popup content. In many ways this method is similar to the .open() method. You can pass other properties in the options object to the popup. They will be applied on top of properties retrieved from the server. When you are loading content from the server, you can optionally add #id at the end of your url. This allows to select a particular popup element by id, if server returns multiple in the same file.
