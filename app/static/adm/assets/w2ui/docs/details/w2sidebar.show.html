Shows sidebar node or nodes with id.
show(id1, [id2], ...)
id string, id of the sidebar node
Returns integer.

Description

The node(s) must exists in the .nodes array of the sidebar or its nested nodes. The node is identified by the .id property. The method will set .hidden property of the node to false. It returns the number of shown nodes that were hidden before.
