Removes event listener from the object.
off(type, [handler])
type string, name of the event
handler function, JavaScript function (optional)
OR
off(eventData, handler)
eventData object, parameters of the event
handler function, JavaScript function
Returns undefined.

Description

You can remove a specific listener for a specific event, all listener for a specific event, or all listener for all events.
