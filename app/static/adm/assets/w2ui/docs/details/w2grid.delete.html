Called when user deletes record(s).
delete(force)
force boolean, indicates if confirmation message is not necessary
Returns undefined.

Description

If force is true, the grid will not display confirmation before deleting records. If you data source is local the records will be deleted from the .records array. If the data source is remote, a delete request will be submitted to the .url. User can select multiple records to delete. The user might delete records by click on the delete toolbar button or by clicking the delete key on the keyboard. This method emits .onDelete event.
