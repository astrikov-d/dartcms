Called when toolbar item is clicked.
onClick = function(event)
You can add event listener during the object creation: or after the object has been created: You can add a listener to a toolbar item, see .items property. If toolbar item is a menu, then the event will have a reference to a menu item user clicked (if any).
The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
