State of the popup.
String, default 'closed'
The best way to find out if popup is opened (and since only one popup can be opened at any particular time), to use jQuery to find popup div and check if it exists: You can also find out what is the state of the popup if you read status property, which will provide you with more information about popup states, such as open, closed, resizing, etc. Status can be one of the following:
  • opening
  • open
  • closing
  • closed
  • resizing
  • moving
