Finds listview item with id and extends it with item object.
set(id, item)
id string, id of the listview item
item object, new listview item object
Returns boolean.

Description

The first argument id is the id of the listview item you used when you created listview items. The .items array holds all created listview items. The listview item object is not replaced with item, but extended by it. If you call this method it during run-time, you will need to call .refresh() method. If listview item was successfully extended the method will return true, otherwise false.
