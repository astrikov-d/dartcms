Called when user pastes from clipboard (ctrl+v or cmd+v).
onPaste = function(event)
The normal behavior is to paste text from the clipboard. You need to have editable columns and selectType to be 'cell'. You can add event listener during the object creation: or after the object has been created: The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
