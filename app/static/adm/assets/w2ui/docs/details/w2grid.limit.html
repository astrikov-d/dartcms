Number of records to return from remote data source per attempt.
Integer, default = 100
This property only applies if your data source for the grid is remote. As of 1.3, the default behavior of the grid is infinite scroll, which means as user scrolls more records will be loaded. Each time a request will be sent to the server to retrieve records, it will submit .limit as the number of records to return.
