Removes nodes from the sidebar.
remove(id1, id2, ...)
id string, id of the sidebar node
Returns integer.

Description

Removes a sidebar node or nodes from the .nodes array and refreshes the sidebar. The sidebar node is identified by the .id property. it will search the entire sidebar including all its nested nodes. The method returns the number of nodes it removes.
