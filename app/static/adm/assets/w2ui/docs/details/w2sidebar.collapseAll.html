Collapses all expanded children of specified node.
collapseAll([parent])
parent string, id of the sidebar node, (optional)
Returns undefined.

Description

The node must exists in the .nodes array of the sidebar or its nested nodes. The node is identified by the .id property. The method will set .expanded property of all children of the node to false. If you do not specify parent argument, then all expanded nodes of the entire sidebar will be collapsed.
