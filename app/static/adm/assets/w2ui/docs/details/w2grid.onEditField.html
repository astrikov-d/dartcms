Called when user enters into editable mode for the field.
onEditField = function(event)
This event will only be triggered if you have defined editable columns. You can add event listener during the object creation: or after the object has been created: The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
