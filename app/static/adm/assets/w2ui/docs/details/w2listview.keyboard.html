Indicates if listview should listen to keyboard.
Boolean, default = true
If .keyboard property is set to true, the listview will add event listener to listen for keyboard events. There are several keyboard keys linked to the listview actions:
  • Arrow up - upper item
  • Arrow down - down item
  • Arrow right - next item
  • Arrow left - previous item
This property can be defined during the listview creation: Or any time after the listview has been created: Internally, the feature is implemented by w2utils.keyboard object. If there more then one listview on the screen and all of them have .keyboard set to true, then the last listview where user selects a record will active keyboard listener.
