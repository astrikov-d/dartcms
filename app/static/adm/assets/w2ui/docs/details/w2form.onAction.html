Called when user executes a form action.
onAction = function(event)
If you have .actions defined for the form, each time an action gets executed it will trigger this event. You can add event listener during the object creation: or any time during run-time: The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
