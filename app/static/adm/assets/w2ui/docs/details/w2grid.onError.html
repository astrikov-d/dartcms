Called when server returns an error.
onError = function(event)
This event is triggered each time an error message needs to be displayed. In most cases the error message is displayed when server returns an error, but you can call .error() method on your own errors and it will trigger error event.
You can add event listener during the object creation: or after the object has been created: Your event listener can be executed before or after the event has been processed. You can also cancel the default behavior. See Events page in utilities.
